# issue-sync

## Developing

```bash
# Restore dependencies
yarn install

# Spin up local lambda test server
yarn run dev

# Test endpoints by calling them in your browser,
# or with a tool like Postman
# Navigate to: `http://localhost:9000/.netlify/functions/hello_world`
```
